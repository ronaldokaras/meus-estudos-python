## 📌 Sobre o projeto

O **Super Caixa** é um sistema simples de checkout feito no terminal que simula o funcionamento básico de um caixa de loja.

O programa permite registrar vendas, escolher a forma de pagamento e aplicar descontos ou juros automaticamente.

## ⚙️ Funcionalidades

- Aplicação de **15% de desconto para pagamento via Pix**
- Aplicação de **5% de desconto para pagamento no Débito**
- Pagamento no **Crédito à vista sem juros**
- **Parcelamento no crédito com 10% de juros**
- Sistema contínuo utilizando **loop `while`** para múltiplas vendas

## 💻 Tecnologias utilizadas

- Python 3.x
- Terminal / CLI


## 🧪 Exemplos de execução
Cenário 1 — Pagamento no Pix
Valor da Compra: R$ 150
Escolha a forma de Pagamento (1 - Pix, 2 - Débito, 3 - Crédito): 1

Sua compra teve 15% de desconto
Valor Total: R$ 127.50

Deseja realizar outra venda? (s/n): n

Caixa fechado. Até amanhã!
Cenário 2 — Pagamento no Débito
Valor da Compra: R$ 200
Escolha a forma de Pagamento (1 - Pix, 2 - Débito, 3 - Crédito): 2

Sua compra teve 5% de desconto

Cenário 3 — Crédito parcelado com juros
Valor da Compra: R$ 300

Total com juros: R$ 330.00
Você parcelou sua compra em 3x e ficou: R$ 110.00 por parcela
